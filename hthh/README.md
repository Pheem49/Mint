# HBD Project for Friend 🎂

โปรเจกต์เว็บไซต์วันเกิดแบบเรียบหรูและเป็นกันเอง สร้างขึ้นเพื่อส่งต่อความสุขให้เพื่อน

## โครงสร้างไฟล์
- `index.html`: โครงสร้างหน้าเว็บหลัก
- `styles.css`: การตกแต่งและดีไซน์แบบ Responsive
- `script.js`: ระบบแอนิเมชันและเอฟเฟกต์พลุ (Confetti)

## วิธีใช้งาน
1. เปิดไฟล์ `index.html` ด้วยบราวเซอร์
2. กดปุ่ม "ฉลองเลย! 🎉" เพื่อดูเอฟเฟกต์พลุ
3. คุณสามารถเปลี่ยนข้อความในไฟล์ `index.html` เพื่อปรับแต่งคำอวยพรได้ตามต้องการ

## เทคโนโลยีที่ใช้
- HTML5 / CSS3
- JavaScript (Vanilla)
- Canvas Confetti Library (via CDN)
